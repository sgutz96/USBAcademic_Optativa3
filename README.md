## Descripción del Curso

Este curso está diseñado para integrar conceptos avanzados de **diseño UX/UI**, **programación web** y **desarrollo de entornos 3D y VR**. Los estudiantes aplicarán sus conocimientos en el desarrollo de aplicaciones, creación de experiencias de usuario interactivas, gamificación, y realidad virtual (VR). A lo largo del curso, se explorarán tecnologías como **Vue.js**, **React**, **Unity**, **WordPress**, y **Elementor**.

## Estructura del Curso

El curso se divide en tres cortes con un enfoque progresivo:

## Actividades y Requisitos Técnicos

### Actividades de Diseño:
- **Figma**: Creación de prototipos, análisis heurísticos, pruebas de usabilidad, diseño de interfaces responsivas y accesibles.
- **Pruebas de Usabilidad**: Implementación de pruebas A/B, pruebas de calor, y análisis de la interacción del usuario con herramientas como **Hotjar** y **Crazy Egg**.
- **Diseño Web y UX/UI**: Uso de **WordPress** y **Elementor** para diseñar interfaces web accesibles y responsivas.

### Actividades de Programación:
- **Vue.js**: Desarrollo de aplicaciones con componentes, gestión de estado, navegación, y pruebas de usabilidad.
- **React**: Creación de aplicaciones interactivas con **React**, manejo del estado con **useState** y **useEffect**, consumo de APIs, y desarrollo de **SPA**.
- **Unity**: Creación de mundos 3D y entornos **VR**, desarrollo de interacciones, programación de HUDs, y gamificación.
